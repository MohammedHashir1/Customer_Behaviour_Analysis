Customer Behavior Analysis
Overview

This project analyzes customer shopping behavior to uncover insights into purchasing patterns, customer preferences, and overall trends. The goal is to support data-driven decision-making and improve business strategies.

Dataset
Total Records: 3,900
Total Features: 18
Includes:
Customer demographics (age, gender, location, subscription status)
Purchase details (items, category, amount, season, size, color)
Shopping behavior (discounts, promo codes, frequency, ratings, shipping type)
Tools & Technologies
Python (pandas, numpy)
SQL (PostgreSQL / MySQL)
Power BI
Gamma (for presentation)
Steps
Data Loading using pandas
Data Cleaning and handling missing values
Exploratory Data Analysis (EDA)
Feature Engineering (age groups, purchase intervals)
Data consistency checks and column standardization
SQL queries for deeper insights
Dashboard creation in Power BI
Presentation creation using Gamma
Dashboard

An interactive Power BI dashboard was built to visualize:

Customer segments
Sales trends
Product performance
Purchase behavior patterns
Results
Identified key purchasing trends and customer segments
Improved understanding of product performance
Discovered patterns in discounts and customer behavior
Generated actionable business insights
How to Run
Clone the repository
Install required Python libraries (pandas, numpy)
Load the dataset in Python
Run data cleaning and EDA scripts
Execute SQL queries in PostgreSQL/MySQL
Open the Power BI file to explore the dashboard# Customer_Behaviour_Analysis
Analyzed customer behavior using Python, SQL, and Power BI. Performed data cleaning, feature engineering, and EDA on 3,900 records to identify purchasing patterns and trends. Generated insights to improve targeting, promotions, and data-driven decision-making.
